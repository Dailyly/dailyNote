# atom功能介绍

## 快捷键
- 关闭标签页  
**win:** `Ctrl + w`

- 左侧树状菜单的显示和隐藏  
**win:** `Ctrl + \`

- 选择整行  
**win:** `Ctrl + L`

- 控制台  
**win:** `Ctrl + Shift + P`

- markdown 文本预览  
**win:** `Ctrl + Shift + P` 输入 `markdown Preview: Toggle`

- 查找整个项目
**win:** 右键左侧树状菜单中的项目文件夹 `Search in Directory`


# 常用的插件
## active-power-mode
    炫酷的编辑动画效果，右侧显示combo的数字。combo超过50的时候，整个屏幕都会动。
    但是感觉比较占内存或者cpu，可以选择性的关闭。

## emmet/emmrt-atom
    每一个前端必备，编写html非常好用

## file-icons
    修改默认图标的样式
