# 知识点
## 日期
```js
Date.now()      // 获取当前时间毫秒数
var dt = new Date()
dt.getTime()    // 获取毫秒数   dt对象创建时候的毫秒数
dt.getFullYear()    // 年
dt.getMonth()       // 月(0-11)
dt.getDate()        // 日(0-31)
dt.getHours()       // 小时(0-23)
dt.getMinutes()     // 分钟(0-59)
dt.getSeconds()     // 秒(0-59)
```

## 数组API
- forEach 遍历所有元素
- every 判断所有元素是否都符合条件
- some 判断是否有至少一个元素符合条件
- sort 排序
- map 对元素重新组装，生成新数组
- filter 过滤符合条件的元素


### forEach 遍历所有元素
```js
var arr = [1,2,3]
arr.forEach(function (item, index) {
    // 遍历数组的所有元素
    console.log(index, item)
})
```
